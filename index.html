Git - repository
  • git clone <ссылка на репозиторий>: Скачать проект к себе
  • git status: статус текущего локального GITa
  • git add path-to-file: добавить файл под контроль GITa
  • git add --all: добавить все файлы под контроль GITa
  • git commit -am "comment": зафиксировать изменения и написать комментарий
  • git push: отправить все изменения в репозиторий на сервер
  • git pull: забрать свежие изменения из репозитория
  • git checkout <имя ветки>: Сменить ветку
